# OrgChart Project

Template project with frontend (React + Tailwind) and backend (Express + MySQL).

## How to run

```bash
# Backend
cd backend
npm install
cp .env.example .env
npm run dev

# Frontend
cd frontend
npm install
npm run dev

# Docker (optional)
docker-compose up --build
```

## require library
d3-org-chart
jspdf
html2canvas
